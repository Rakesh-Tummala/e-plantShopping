# e-plantShopping

Paradise Nursery is a React and Redux based shopping application for indoor houseplants.

## Features

* Landing page with company information
* Product listing page with categorized plants
* Shopping cart functionality using Redux
* Add, remove, increase, and decrease cart items
* Dynamic cart counter in the navigation bar
* Responsive user interface

## Technologies Used

* React
* Redux Toolkit
* JavaScript
* CSS
